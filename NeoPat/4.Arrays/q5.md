Given an array of integers, find the majority element. If no majority element is present, then display 0.



Note: A majority element is an element in an array that occurs more than (size/2) times in the array.

Input format :
The first line of input consists of the size of the array.

The second line of input consists of the array of elements separated by spaces.



Refer to the sample input for formatting specifications.

Output format :
The output prints the "majority element" of the array.

Otherwise, print 0.



Refer to the sample output for formatting specifications.

Sample test cases :
Input 1 :
9
3 3 4 2 4 4 2 4 4
Output 1 :
4
Input 2 :
8
3 3 4 2 4 4 2 4
Output 2 :
0
