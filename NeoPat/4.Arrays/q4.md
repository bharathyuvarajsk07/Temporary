Given an array of integers, find all the leaders in the array. Display the count of leaders in the given array.



Note: An element is a Leader if it is greater than all the elements to its right side. The rightmost element is always a Leader.

Input format :
The first line of input consists of the size of the array.

The second line of input consists of the array of elements separated by spaces.



Refer to the sample input for formatting specifications.

Output format :
The output displays the count of the leaders in the given array.



Refer to the sample output for formatting specifications.

Sample test cases :
Input 1 :
6
16 17 4 3 5 2
Output 1 :
3
Input 2 :
6
2 5 8 7 3 6
Output 2 :
3
